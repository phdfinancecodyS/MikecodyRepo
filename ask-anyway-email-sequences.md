# HOW TO SET THIS UP IN CONVERTKIT (15 minutes)

1. Go to **app.convertkit.com → Automations → Sequences**
2. Click **New Sequence** → name it (e.g. "Ask Anyway — Low Risk")
3. Add each email above as a new email in the sequence, set the delay days
4. Click **Save**
5. Go to **Automations → Visual Automations → New Automation**
6. Trigger: **"Tag is added" → Low Risk**
7. Action: **"Subscribe to sequence" → Ask Anyway — Low Risk**
8. Repeat for all 4 tags
9. Activate each automation

That's it. Every quiz submission will automatically route into the right sequence.
