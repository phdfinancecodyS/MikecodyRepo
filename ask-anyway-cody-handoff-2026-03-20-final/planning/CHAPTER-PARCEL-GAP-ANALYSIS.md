# Chapter Parcel + Gap Analysis (Human-Voice Format)

Date: 2026-03-20
Source set: 47 current chapter titles in frontline guide catalog

## Decisions Locked

1. Keep human voice: plain, direct, lived-language, no academic framing.
2. Every guide includes 2 worksheets minimum.
3. Parcel out multi-topic chapters into tighter buyer-intent guides where needed.

## How To Parcel (Rule of Thumb)

Split a chapter if it includes 2+ of these in one title:
- multiple symptom clusters
- multiple audiences (self + partner + team)
- multiple life contexts (home + work + body)
- multiple intervention types (awareness + script + plan)

Keep as one guide if one clear user intent solves one moment.

## Writing Guardrails (Keep It Human)

Use:
- "Here is what to say"
- "If this is your week, start here"
- "Do this in the next 10 minutes"

Avoid:
- diagnosis labels as the primary frame
- theory-heavy explanation blocks
- academic citation voice inside the main body

## Worksheet Standard (Apply To Every Guide)

Every guide ships with exactly 2 worksheets minimum:

Worksheet A: Awareness
- identify pattern, trigger, or bottleneck
- produce one clear insight

Worksheet B: Action
- produce one script and one 24-hour action plan
- include checkbox commitments

If a chapter is split into multiple guides, each resulting guide still gets its own 2 worksheets.
