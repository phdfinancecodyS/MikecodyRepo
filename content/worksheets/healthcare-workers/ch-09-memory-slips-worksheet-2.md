# Memory Slips: Action Builder

**Audience:** Healthcare Workers

**From guide:** Memory Slips

---

Goal: Turn what you noticed into one action and one commitment.

Prompts:
- One external memory tool I'll start using today:
- The three priorities I'll write down tonight for tomorrow:
- One thing I'll take off my plate to reduce cognitive load:
- The person I'll tell about the memory issues so they can help:
- By tomorrow night, I'll know this worked if:

---

**A quick note:** This worksheet is educational: it's not therapy, it's not a diagnosis, and using it doesn't create a clinical relationship between us. It's built to help you take the next step, not replace professional support. If you're in crisis, contact 988 (Suicide & Crisis Lifeline), text HOME to 741741 (Crisis Text Line), or call 911.
