Ask Anyway Setup Guide

Get the chatbot + backend running from scratch on any Mac. Estimated time: ~10 minutes.

What You Need

  • macOS (Monterey or later)
  • Python 3.11+ - check with python3 --version
    Install from python.org if missing
  • Git - check with git --version
  • Internet access (for therapist lookups)

Step 1 - Get the Code

1
Clone the repo

Open Terminal and run:

cd ~/Desktop
git clone https://github.com/phdfinancecodyS/MikecodyRepo.git ask-anyway
cd ask-anyway

You should now have a folder called ask-anyway on your Desktop.

Step 2 - Set Up the Backend

1
Create a Python virtual environment
cd ~/Desktop/ask-anyway/cce-backend
python3 -m venv .venv
2
Activate it and install dependencies
source .venv/bin/activate
pip install -r requirements.txt

This installs FastAPI, the conversation engine, and the therapist-finder scraper. Takes about 1–2 minutes.

3
Start the backend server
CCE_CORS_ORIGINS="http://localhost:3131" python3 launcher.py

You should see:

INFO:     Uvicorn running on http://0.0.0.0:8000 (Press CTRL+C to quit)
INFO:     Application startup complete.
✓ Backend is running on port 8000. Leave this terminal open.

Step 3 - Serve the Chatbot

1
Open a second Terminal tab

Press ⌘ T in Terminal to open a new tab, then:

cd ~/Desktop/ask-anyway
python3 -m http.server 3131
✓ File server running on port 3131. Leave this open too.

Step 4 - Open the App

1
Open in your browser

Go to: http://localhost:3131/ask-anyway-chat.html

You should see the Ask Anyway chatbot. Type anything to start.

What to Expect

InputWhat Happens
"I'm doing great"GOOD track - 2 warm turns, then closes
"I'm not okay / anxious / going through a breakup"NOT-GOOD track - 3 empathetic turns, then resource cards + guide PDFs
"I've been drinking every night to cope"RISKY track - non-judgmental probe, then resources
"I don't want to be here anymore"CRISIS - immediate crisis card with 988 hotline

Therapist Finder

On any resource card screen, tap "Find Therapists Near Me". Allow location or enter a ZIP code - it will pull real therapists from Psychology Today matched to the conversation topic.

Guide PDFs

Two free PDF guides are linked from resource cards:

  • Connection 101
  • Daily Mental Health Checklist

Verify Everything is Working

Open a third Terminal tab and run the beta test suite:

cd ~/Desktop/ask-anyway
.venv/bin/python3 beta_test.py 2>/dev/null

Wait ~60 seconds (it scrapes Psychology Today for each topic). You should see 46/46 PASS.

⚠ The therapist scraper makes live web requests - results depend on internet access and Psychology Today availability.

Troubleshooting

Port already in use

# Kill whatever is on port 8000
lsof -ti:8000 | xargs kill -9

# Kill whatever is on port 3131
lsof -ti:3131 | xargs kill -9

Therapist cards show 0 results

Psychology Today blocks scrapers occasionally. Wait a minute and try again, or try a different ZIP code.

Backend won't start

cd ~/Desktop/ask-anyway/cce-backend
source .venv/bin/activate
pip install -r requirements.txt   # re-run install
python3 launcher.py

Page shows but chatbot doesn't respond

The backend (port 8000) must be running. Check localhost:8000/health - it should return {"status":"ok"}.
