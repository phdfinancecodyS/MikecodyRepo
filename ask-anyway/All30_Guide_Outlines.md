# PRODUCTION NOTES FOR PARTNER

## Writing Style for All Guides
- **Voice:** Warm, direct, non-clinical. Like a knowledgeable friend, not a textbook.
- **Length:** Each guide = 8-15 pages in Canva. Enough to feel substantial, not overwhelming.
- **Format:** Numbered sections, bullet points, bold key phrases, white space. Easy to skim.
- **Tone:** Validating first, informative second. Every guide opens with "you're not alone."
- **Links:** All resource links verified before publishing. No dead links.

## Safe Messaging Requirements (Non-Negotiable)
- No method details for self-harm or suicide in any guide
- No before/after weights, calories, or numbers in eating disorder guide
- No graphic violence descriptions in DV guide
- Crisis resources always appear BEFORE any paywall
- Every guide includes: "This guide is not a substitute for professional care."

- Guide designed in Canva → exported as PDF
